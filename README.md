# Django Backend – Gaming Social Media Platform

This is the **backend API** for a gaming‑focused social media platform built with **Django**, **Django REST Framework**, **JWT authentication (cookies)**, and **Django Channels** for real‑time chat.

It uses a **Next.js frontend** and provides:

* User authentication (JWT in HttpOnly cookies)
* Gamer profiles & game‑based search
* Posts, comments, likes
* Friend requests
* Real‑time chat (WebSockets)

---

##  Tech Stack

* **Python 3.10+**
* **Django 4+**
* **Django REST Framework**
* **SimpleJWT** (JWT auth)
* **django‑cors‑headers** (CORS + credentials)
* **Django Channels** (WebSockets)
* **Redis** (channel layer)
* **PostgreSQL** (recommended)
